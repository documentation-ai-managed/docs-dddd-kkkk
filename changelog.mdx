## Upgrading to the Latest Version

Follow these steps to update your Dddd Kkkk instance seamlessly.

<Steps>
  <Step title="Backup Your Data" icon="database">

  Export your documentation projects to avoid data loss.

  ```
  # Run backup command
  dddd-kkkk backup --output ./backups/latest.tar.gz
  ```

  </Step>
  <Step title="Update Package" icon="download">
    <CodeGroup tabs="npm,yarn">
      ````bash
      npm install dddd-kkkk@latest
      ````

      ````bash
      yarn add dddd-kkkk@latest
      ````
    </CodeGroup>
  </Step>
  <Step title="Run Migrations" icon="settings">
    ```
    dddd-kkkk migrate --env production
    ```
  </Step>
  <Step title="Restart Service" icon="refresh-cw">
    Restart your server and verify at `https://dashboard.example.com`.
  </Step>
</Steps>

<Callout kind="tip">
  Test upgrades in a staging environment first. Review breaking changes in each release.
</Callout>
